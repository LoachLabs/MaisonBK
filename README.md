# MaisonBK

The website behind [MaisonBK](http://maisonbk.ca).


## Dependencies

* [Node.js](http://nodejs.org)
* [npm](http://npmjs.com)


## Usage

### Install Dependencies

```
npm install
```

### Build

```
npm test
```

### Develop

```
npm start
```

### Deploy

```
npm run deploy
```
